 **User Story**
As a registered user, I want to be able to upload and update my profile picture, so that my profile reflects my personal brand and is easily recognizable.

**Description**
As a registered user, I should be able to upload and update my profile picture in my account settings. This feature will allow me to personalize my profile and make it easily recognizable to other users.

**Acceptance Criteria**

1. When I am logged into my account, I should see an option to edit my profile settings.
2. In the profile settings, I should see an option to upload or change my profile picture.
3. I should be able to select a file from my device to upload as my profile picture.
4. The system should validate the file type and size, and display an error message if the file is not valid.
5. After a successful upload, the system should display a preview of the new profile picture.
6. I should be able to save the changes to update my profile picture.
7. After saving, the system should update my profile picture in all relevant areas of the application.
8. The system should allow me to revert back to my previous profile picture if I change my mind.
9. The system should store my profile picture securely and not make it publicly accessible.
10. The system should display a default profile picture if I have not uploaded one.
11. The system should allow me to delete my profile picture if I choose to do so.
12. After deleting my profile picture, the system should display the default profile picture.
